Este proyecto es una página web básica desarrollada en HTML y CSS como parte de un curso de Front-End. Simula una tienda ecommerce de venta de articulos para perros.
Se han utilizado etiquetas semánticas, los estilos se aplican desde un archivo externo.
Cuenta con un formulario de contacto que utiliza la página Formspree como servidor.
Cuenta con diseño responsive, en donde se ha aplicado FlexBox y Grid.
Las fuentes son de Googlefonts y las imágenes de las hojas 'Productos', 'Accesorios', 'Camas y Cuchas', 'Juguetes' y 'Ropa' fueron creadas con Gemini.
